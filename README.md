# Yunnan Tutor Platform — Demo Deck

Internal pitch demos for a Yunnan-province tutor-finding platform.
Three separate design directions to evaluate before committing to a single concept.

## Live demos

- Overview / landing page → `index.html`
- Demo A — YouShiHui (优师荟): classic marketplace → `demo1-youshihui/index.html`
- Demo B — ShiYuan (师缘): short-video discovery feed → `demo2-shiyuan/index.html`
- Demo C — XueBan (学伴): textbook-QR to tutor matching → `demo3-xueban/index.html`

All content is in Simplified Chinese. Tailwind is loaded via CDN (requires internet).

## Notes

- Static HTML/CSS/JS. No build step. No backend.
- Designed for desktop viewing in Chrome / Edge / Safari.
- Demo B looks best on wider screens (it simulates a phone).
